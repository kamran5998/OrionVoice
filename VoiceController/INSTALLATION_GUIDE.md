# Orion Voice Assistant - Installation Guide

## ðŸ“± Mobile Mein Install Karne Ka Complete Process

### Method 1: Android Studio Se (Recommended)

#### Step 1: Android Studio Setup
1. **Android Studio Download** karo: https://developer.android.com/studio
2. Install karo aur setup complete karo
3. **USB Debugging** enable karo phone mein:
   - Settings > About Phone > Build Number (7 times tap)
   - Settings > Developer Options > USB Debugging (ON)

#### Step 2: Project Build
1. Android Studio open karo
2. **"Open an Existing Project"** select karo
3. Ye folder select karo: `VoiceController`
4. Project load hone ka wait karo
5. **Build > Build Bundle(s)/APK(s) > Build APK(s)**
6. Build complete hone ka wait karo

#### Step 3: Phone Mein Install
1. USB cable se phone connect karo
2. **Run > Run 'app'** ya Green Play button press karo
3. Phone select karo list mein
4. App automatically install ho jayega

---

### Method 2: APK File Banake (Easy Method)

#### Step 1: APK Generate
1. Android Studio mein project open karo
2. **Build > Generate Signed Bundle/APK**
3. **APK** select karo, Next
4. **Create new keystore** (first time)
5. Details fill karo, passwords set karo
6. **release** build type select karo
7. **Finish** - APK ban jayega

#### Step 2: APK Transfer
1. APK file find karo: `app/release/app-release.apk`
2. **WhatsApp/Email/Bluetooth** se phone mein bhejo
3. Ya **USB cable** se direct copy karo

#### Step 3: Phone Mein Install
1. Phone mein APK file open karo
2. **"Install from Unknown Sources"** allow karo
3. **Install** button press karo
4. App install ho jayega

---

### Method 3: Direct USB Transfer (Fastest)

#### Step 1: APK Copy
1. Computer mein APK file ready karo
2. Phone ko USB cable se connect karo
3. **File Transfer** mode select karo
4. APK file phone ke **Downloads** folder mein copy karo

#### Step 2: Install
1. Phone mein **File Manager** open karo
2. **Downloads** folder mein jao
3. APK file pe tap karo
4. **Install** karo
